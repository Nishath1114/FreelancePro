Installation

Clone the repository:

git clone https://github.com/Nishath1114/FreelancePro/blob/main/Freelance-Forge.zip


Navigate to the project directory:

cd FreelancePro


Install dependencies:

npm install   # or pip install -r requirements.txt


Set up environment variables:

# Example
DATABASE_URL=your_database_url
SECRET_KEY=your_secret_key


Run the project:

npm start   # or python manage.py runserver

Usage

Sign up as a freelancer or client

Post projects or browse available ones

Bid on projects, manage tasks, and communicate securely

Process payments and leave reviews
